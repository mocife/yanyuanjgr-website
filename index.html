Project Background

01 项目背景

为什么我们需要燕园主理人？

城市化进程过快，规划决策与实际需求脱节，存量公共空间存在诸多问题。 城市规划复杂且门槛高，公众参与深度不足。 我们相信：规划不是自上而下的蓝图，而是自下而上的对话。

"规划不是自上而下的蓝图，而是自下而上的对话。"

🎯

打破空间壁垒

虚拟城市空间已成为现实空间的镜像与延伸，游戏化媒介是公众参与的有效路径。

🚪

降低参与门槛

严肃游戏融合互动机制与情境模拟，让全年龄段都能参与复杂规划。

⚡

提升参与效率

虚拟界面让居民直接进行空间布局，数据收集效率远高于传统调研。

Game Design

02 游戏设计

《燕园主理人》— 经营类放置装修游戏

🎮 核心玩法

玩家以北大学生的身份，在校园中探索、发现各类空间问题，通过资源管理和创意设计，亲手改造并激活校园空间，提升师生满意度和校园活力。
🗺️ 可探索区域：北京大学景观学院与城环学院园区
⏱️ 游戏时长：15-30分钟完成一个完整循环
📍

定位发现

地理围栏算法 + "你画我猜"式交互验证，在现实中发现地块、解锁空间。

🏗️

网格建造

130+物品库，4大类（公共设施/装饰艺术/景观植被/道路铺装），支持放置、旋转、删除。

🤖

AI NPC

DeepSeek API 驱动的智能体对话 NPC，Raby 活泼兔等你来解锁！

☁️

云端存档

LeanCloud 后端支持，建造数据云端同步，好友校园互联互通。
