01 / About

关于游戏

城市化进程过快，规划决策往往与实际需求脱节。公众参与常流于形式，缺乏深度与实效。

燕园主理人将复杂的社会问题转化为可交互的"副本"——玩家在像素风的燕园中，发现校园问题、提交改造方案、参与社区共建，体验15-30分钟的完整"发现—改造—效益"循环。

"规划不是自上而下的蓝图，而是自下而上的对话。"

02 / Features

核心玩法

网格建造系统

基于网格化规则的3D空间创作系统，在规定的网格内放置、旋转、移除各类组件，构建个性化的虚拟校园。拥有130+可放置物品，涵盖公共设施、装饰艺术、景观植被、道路铺装四大类。

  • ✓ 精确坐标计算，实时物品管理
  • ✓ 绿色/红色预览，智能冲突检测
  • ✓ 作品保存与云端上传

AI 驱动的 NPC

接入 DeepSeek API，为每一个 NPC 赋予独特的性格与对话能力。玩家与 NPC 对话，获取任务指引与校园故事，通过积累触发机制解锁更多互动内容。

  • ✓ DeepSeek API 驱动角色对话
  • ✓ 可调温度值，控制输出随机性
  • ✓ 角色装扮与任务限定系统

位置发现与诊断

采用地理围栏算法构建经纬度矩形包围盒，配合0.0002°容差过滤GPS信号漂移。玩家在现实燕园中寻找线索、比对视角，获得改造灵感与游戏奖励。

  • ✓ GPS + 地理围栏双重校验
  • ✓ "你画我猜"式交互验证
  • ✓ 激励玩家观察场地细节

社交互访与共创

好友校园互访机制，拜访彼此的校园，查看不同的改造方案。NPC AI根据访客的装扮和互动产生差异化反应，云端数据实时收集，为现实营建决策提供支持。

  • ✓ 好友校园互访与参观
  • ✓ NPC差异化互动响应
  • ✓ 云端数据支撑现实决策
